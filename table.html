100 Essential Git Commands with Explanations

This guide presents 100 essential Git commands categorized into different functionalities.

# Command Description
1 git init Initializes a new Git repository.
2 git clone [url] Clones an existing repository from a remote source.
3 git add [file] Stages a specific file for commit.
4 git add . Stages all changes in the current directory.
5 git commit -m "message" Commits the staged changes with a message.
6 git status Shows the current status of the repository.
7 git branch Lists all branches in the repository.
8 git checkout [branch-name] Switches to the specified branch.
9 git merge [branch] Merges the specified branch into the current branch.
10 git push Pushes committed changes to the remote repository.

These are just the first 10 Git commands. More commands are available in different sections.
